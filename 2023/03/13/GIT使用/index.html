GIT使用

基本linux命令

  1. cd 切换目录
  2. cd.. 返回上一级目录
  3. pwd 显示当前所有目录==路径==
  4. ls 列出当前所有文件，白色代表文件，绿色代表可执行文件，紫色代表目录
  5. touch 新建一个==文件== touch index。html
  6. rm 删除文件
  7. mkdir 新建一个==文件夹==
  8. rm-r 删除文件夹
  9. mv 移动文件
  10. clear 清屏
  11. history 查看历史命令
  12. exit 退出
  13. #注释

必要配置

  1. 用户名 git config –global user.name “”
  2. 邮箱 git congig –global user.email “”
