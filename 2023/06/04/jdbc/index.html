第二章获取链接

通过Driver获取链接——————驱动类

2.1要素一：

Driver**接口实现类

java.sql.Driver 接口是所有 JDBC 驱动程序需要实现的接口。这个接口是提供给数据库厂商使用的，不同数据库厂商提供不同的实现。

在程序中不需要直接去访问实现了 Driver 接口的类，而是由驱动程序管理器类(java.sql.DriverManager)去调用这些Driver实现。

2.2 要素二：

****URL

URL 用于标识一个驱动程序，驱动程序管理器通过这个==URL 选择正确的驱动程序==，从而建立到数据库的连接

2.3 要素三：

用户名和密码

获取连接步骤

方式一

  1. 创建一个Driver （驱动）对象
  2. 两个连接函数参数url（不同数据库提供的API） 还有 Property（数据库连接的账户还有密码）
  3. 调用Driver的链接函数 deiver。connect（url，property）

方式二（加了反射）

  1. 利用反射创造Class对象 Class。forName（“com.mysql.cj.jdbc.Deiver”）
  2. 利用class调用newInterstance 创建Driver对象
  3. 两个连接函数参数url（不同数据库提供的API） 还有 Property（数据库连接的账户还有密码）
  4. 调用Driver的链接函数 deiver。connect（url，property）

​ Connection conn = driver.connect(url, info);

方式三（加了驱动管理）

  1. 利用反射创造Class对象 Class。forName（“com.mysql.cj.jdbc.Deiver”）

  2. 利用class调用newInterstance 创建Driver对象

  3. 注册驱动 ==静态方法==参数是driver对象

    ​ 调用DriverManager静态方法 DriverManager.registerDriver()

  4. 用驱动管理调用连接方法 ==静态方法== 参数 url, user, password直接输入进去就行

    ​ Connection conn = DriverManager.getConnection(url, user, password);

方式四

​ 原因在创建Class对象时，mysql提供的Api已经编写了创建Driver对象和注册驱动代码

  1. 利用反射创造Class对象 Class。forName（“com.mysql.cj.jdbc.Deiver”）

  2. 用驱动管理调用连接方法 ==静态方法== 参数 url, user, password直接输入进去就行

    ​ Connection conn = DriverManager.getConnection(url, user, password);

方式五（从文件里读取url，root，password）

  1. 建立输入流

  2. 创建Property，对象

  3. 从文件里读取url，root，password

  4. 利用反射创造Class对象 Class。forName（“com.mysql.cj.jdbc.Deiver”）

  5. 用驱动管理调用连接方法 ==静态方法== 参数 url, user, password直接输入进去就行

    ​ Connection conn = DriverManager.getConnection(url, user, password);

增删改步骤

  1. 建立数据库连接
  2. 编译sql语句 生成PrepareStament实例ps（这个实例代表我们想做的事情
  3. 填充占位符 ps。setObject（1，“莫扎特”）
  4. 执行实例ps.execute()
  5. 关闭资源

单个表查询步骤

  1. 建立数据库

  2. 创建sql 预编译实例ps

  3. 补充占位符

  4. 创建结果集实例rs=ps.executeQuery()，并且创立结果集元数据rsm=rs.getMetaData()（可以获得数据的列数 还有列名）

  5. 判断结果集不为空后，创建对象，获取列值rs.getObject( i+1) ，用rsm.getColumnName( i+1)获得列名

    再通过反射获得所需Field字段，用field.set（对象名，列值）填充对象

  6. 返回对象值

  7. 关闭资源

  1. 建立数据库

  2. 创建sql 预编译实例ps

  3. 补充占位符

  4. 创建结果集实例rs=ps.executeQuery()，并且创立结果集元数据rsm=rs.getMetaData()（可以获得数据的列数 还有列名）

  5. 判断结果集不为空后，创建对象，获取列值rs.getObject( i+1) ，用rsm.getColumnLabel ( i+1)获得列名
