print和sys.stdout.write区别

print 和 sys.stdout.write 都是 Python 中用于输出内容的方法，但它们有一些不同之处：

  1. print 是一个内置函数，而 sys.stdout.write 是 sys 模块中的方法。因此，print 可以直接在代码中使用，而 sys.stdout.write 需要先导入 sys 模块才能使用。
  2. print 可以同时输出多个值，自动以空格分隔，而 sys.stdout.write 只能输出一个字符串，不能自动添加分隔符。 例子 print(12,32,43,54);
  3. print 会在输出内容的末尾添加一个换行符，而 sys.stdout.write 不会自动添加换行符，需要手动添加 \n 才能实现换行。

多态时的方法

  1. 如果一个方法在父类中定义，而子类对该方法进行了重写（即覆盖），当通过父类引用调用该方法时，实际上会调用子类中重写的方法。这就是动态多态的体现。
  2. 如果一个方法在子类中定义，并且没有在父类中定义，那么只有通过子类的引用才能调用这个方法。如果通过父类的引用调用该方法，会导致编译错误。

多态时的属性

  1. 如果一个属性在父类中定义，而子类对该属性进行了重写（即覆盖），当通过父类引用调用该属性时，==实际上会调用子类中重写的属性==。这就是动态多态的体现。

类的私有方法

__private_method：两个下划线开头，声明该方法为私有方法，只能在类的内部调用 ，不能在类的外部调用。self.__private_methods。

有下划线的方法有什么不同

方法名以双下划线 __ 开头和结尾的方法是 Python 中的特殊方法，也称为魔术方法或者魔术属性。这些 特殊方法在类中有特殊的用途

==它们会在特定情况下自动调用，而不需要我们手动调用。==

例如：

  1. __init__: 在创建对象时自动调用，用于对象的初始化。
  2. __str__: 当使用 print(obj) 或 str(obj) 函数时自动调用，返回对象的字符串表示。
  3. __add__: 当使用 + 运算符进行加法操作时自动调用，实现对象的加法运算。
  4. __sub__: 当使用 - 运算符进行减法操作时自动调用，实现对象的减法运算。
  5. __mul__: 当使用 * 运算符进行乘法操作时自动调用，实现对象的乘法运算。
  6. __eq__: 当使用 == 运算符进行相等比较时自动调用，实现对象的相等比较。

Python3 标准库概览

  • os 模块：os 模块提供了许多与操作系统交互的函数，例如创建、移动和删除文件和目录，以及访问环境变量等。
  • sys 模块：sys 模块提供了与 Python 解释器和系统相关的功能，例如解释器的版本和路径，以及与 stdin、stdout 和 stderr 相关的信息。
  • time 模块：time 模块提供了处理时间的函数，例如获取当前时间、格式化日期和时间、计时等。
  • datetime 模块：datetime 模块提供了更高级的日期和时间处理函数，例如处理时区、计算时间差、计算日期差等。
  • random 模块：random 模块提供了生成随机数的函数，例如生成随机整数、浮点数、序列等。
  • math 模块：math 模块提供了数学函数，例如三角函数、对数函数、指数函数、常数等。
  • re 模块：re 模块提供了正则表达式处理函数，可以用于文本搜索、替换、分割等。
  • json 模块：json 模块提供了 JSON 编码和解码函数，可以将 Python 对象转换为 JSON 格式，并从 JSON 格式中解析出 Python 对象。
  • urllib 模块：urllib 模块提供了访问网页和处理 URL 的功能，包括下载文件、发送 POST 请求、处理 cookies
